# Issue Tracker API

A simple Issue Tracker REST API built using Spring Boot.

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Features

- Create Issues
- View Issues
- Store issues in database

## API Endpoints

GET /issues  
Returns all issues

POST /issues  
Creates a new issue

## Database

This project uses H2 in-memory database.

H2 Console:
http://localhost:8080/h2-console

## Swagger Documentation

http://localhost:8080/swagger-ui.html
